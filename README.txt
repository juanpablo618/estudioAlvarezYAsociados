Reglas del negocio:

en turnos es 
realizado = si = verde
realizado = pendiente = Amarillo
realizado = no = nada ningún color 

En agendas es 
realizado = si = verde
realizado = reagendada = amarillo
realizado = nada ningún color

En expedientes es:
color azul = exp. judicial
color blanco = exp. administrativo
color crema = exp. sin carpeta


por el momento todavía no se puede pasar de un exp. sin carpeta a un exp. administrativo o judicial

el cambio que acepta el sistema, es de adm. a judicial. o también al revés.
